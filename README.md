# 💻 About
## TimeGrid is a collaborative historical platform designed for everyone. In this app people can add historical events for people to learn. The events can be fixed or deleted by the user, if something is incorrect. The app overall is really usefull for people that have started learning history or want to enrich their knowledge in history.
